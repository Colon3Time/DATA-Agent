  ---
  Engine (orchestrator.py) — 1,687 บรรทัด

  User input
    ↓
  Anna (DeepSeek) — รับ CLAUDE.md + Anna KB + Session Memory + Agent Specs (RAG)
    ↓ parse <DISPATCH> tags
  run_agent() — load agents/X.md + KB RAG (top 6) + hardcoded rules
    ↓
  DeepSeek เขียน script → รัน subprocess → auto-fix 15 รอบ → Claude fix → ASK_USER
    ↓
  validate_agent_output() — warn ถ้า output ว่าง
    ↓ pipeline_write() ส่ง path ต่อ
  Anna summary → save_session_memory() → auto-continue loop (max 10)

  Features ที่มีแล้ว:
  - RAG KB retrieval (TF-IDF top 6)
  - Agent specs RAG (เลือก 6 agents ที่ relevant)
  - Auto-fix DeepSeek 15 รอบ + Claude fallback
  - Step mode (ถามยืนยันก่อนรัน agent ถัดไป)
  - ESC หยุด script กลางคัน
  - Parallel execution (parallel_group)
  - Session memory ข้าม session
  - Output validation
  - status / resume / kb / claude commands
  - CRISP-DM loop guard (max 5 iterations/agent)
  - KB auto-consolidation ทุก 10 projects
